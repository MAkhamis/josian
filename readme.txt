Webagency Widget allows subscribers to build their own websites through the plugin Custom CMS and customization template.

== Description ==

Add a multi-website builder to your wordpress blog. Offer both free and premium websites to your subscribers. Extremely easy to use. Create as many sites as you like and manage them within WordPress admin.

Major features in Webagency Widget:

* Jquery, ajaxed powered CMS.
* Create unlimited websites.
* Shop for domain.
* Responsive and Customizable template.


== Installation ==

Upload the Webagency Widget plugin to your blog, Activate it. Add Webagency Widget to your sidebar.

1, 2, 3: You're done!
